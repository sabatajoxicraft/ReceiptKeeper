HOW TO GET YOUR APK:
-----------------------------------------------

1. Check the build status:
   gh run list --workflow=build-android.yml

2. Wait for build to complete (takes 5-10 minutes)

3. View the run details:
   gh run view

4. Download the APK artifact:
   gh run download

   OR visit:
   https://github.com/sabatajoxicraft/AppCounterApp/actions

5. Install on your phone:
   - The APK will be in the downloaded folder
   - Transfer to your phone
   - Open and install


MONITOR BUILD PROGRESS:
