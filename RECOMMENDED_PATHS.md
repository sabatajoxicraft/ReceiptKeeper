# Recommended Storage Paths

## For Receipt Images:
```
/storage/emulated/0/Download/ReceiptKeeper
```

## For Log Files:
```
/storage/emulated/0/Download/ReceiptKeeper/Logs
```

## For OneDrive:
```
/Accountant/Receipts
```
(or any folder you want in your OneDrive)

---

Just copy-paste these paths into the Setup screen text fields!
